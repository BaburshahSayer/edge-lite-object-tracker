# EdgeLite: Lightweight Real-Time Object Detection and Tracking

This project aims to design and implement a lightweight object detection and tracking system that can run efficiently on resource-constrained devices such as Raspberry Pi, Jetson Nano, or mobile platforms.

## 🚀 Objectives

- Real-time object detection using lightweight models (e.g., Tiny-YOLO, MobileNet-SSD, YOLOv5n).
- Efficient object tracking using algorithms like SORT, DeepSORT, or ByteTrack.
- Deployable on edge devices with limited compute power.
- Optimize for speed, accuracy, and low power consumption.
- Evaluate trade-offs between performance and efficiency.
